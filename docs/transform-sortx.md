# SortX

对 x 比例尺的定义域（Domain）进行排序，一般用于离散的比例尺（序数比例尺）。可以理解为将几何元素按照 x 通道的值进行分组，然后根据指定的通道和聚合函数进行排序。目前支持的聚合函数如下：

- _max_ - 根据每组的最大值排序
- _min_ - 根据每组的最小值排序
- _a function_ - 根据该聚合函数返回的值排序
