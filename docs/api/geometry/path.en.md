Path 几何标记。<br />用于绘制路径图等。<br />

<a name="47c96d01"></a>

## 继承关系

<br />↳ [Geometry](geometry)<br />
<br />↳ **Path**<br />
<br />↳ [Line](line)<br />
<br />↳ [Area](area)<br />

<a name="d3474432"></a>

## 创建方式

```typescript
chart.path();
view.path();
```

<a name="3d0a2df9"></a>

### 参数

<br />• **connectNulls**? : _boolean_<br />
<br />是否连接空值<br />
<br />• **sortable**? : _boolean_<br />
<br />是否对数据进行排序<br />
<br />• **theme**? : _object_<br />
<br />主题配置<br />
<br />• **visible**? : _boolean_<br />
<br />是否可见
