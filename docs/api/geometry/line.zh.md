Line 几何标记。<br />常用于折线图的绘制。<br />

<a name="b821e2f0"></a>

## 继承关系

<br />↳ [Path](path)<br />
<br />↳ **Line**<br />

<a name="d3474432"></a>

## 创建方式

```typescript
chart.line();
view.line();
```

<a name="3d0a2df9"></a>

### 参数

<br />• **connectNulls**? : _boolean_<br />
<br />是否连接空值<br />
<br />• **sortable**? : _boolean_<br />
<br />是否对数据进行排序<br />
<br />• **theme**? : _object_<br />
<br />主题配置<br />
<br />• **visible**? : _boolean_<br />
<br />是否可见
