# 概述

将抽象数据映射为视觉数据。

## 通用 API

| 参数      | 说明                                                  | 类型                  | 默认值 |
| --------- | ----------------------------------------------------- | --------------------- | ------ |
| formatter | 格式化坐标轴刻度的函数，支持 [d3-format](https://github.com/d3/d3-format) 所定义的字符串。 | `string \| Formatter` | `d => d + ''`   |
